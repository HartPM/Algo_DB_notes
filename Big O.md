# Algo_DB_notes

Big O
- O(1)
- O(log(n))
- O(n)
- O(n log(n))
- O(n^2)

Big O of Objects
- Insertion: O(1)
- Removal: O(1)
- Searching: O(n)
- Access: O(1)

Big O of Arrays
- Insertion: It depends...  [ push/pop: O(1); unshift/shift: O(n) ]
- Removal: It depends...
- Searching: O(n)
- Access: O(1)

- push/pop: O(1)
- shift/unshift: O(n)
- concat: O(n)
- slice: O(n)
- splice: O(n)
- sort: O(n log(n))
- forEach/map/filter/reduce/etc: O(n)
